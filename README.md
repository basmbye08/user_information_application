User Information Application
Project Overview

The User Information Application is a simple Android application developed using Java and XML in Android Studio. The app allows users to enter their personal details such as username, password, address, and age through input fields. After the user submits the form, the application retrieves the entered information and displays it.

This project demonstrates the fundamental concepts of Android development, including user input handling, button click events, and displaying output using Toast messages.

Features

Input fields for:

Username

Password

Address

Age

Submit button to process user information

Displays the entered details using a Toast message

Technologies Used

Programming Language: Java

IDE: Android Studio

UI Design: XML Layouts

Platform: Android SDK

Application Workflow

The user enters their personal details in the input fields.

The user clicks the Submit button.

The application retrieves the text from the input fields.

The information is displayed on the screen using a Toast message.

Learning Objectives

This project helped in understanding:

Working with EditText components

Handling Button click listeners

Retrieving user input using .getText().toString()

Displaying information using Toast messages
